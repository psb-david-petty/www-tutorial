# HTML 5

This document covers HTML 5.

## Minimal HTML 5

```
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>Title</title>
  </head>
  <body>
    <!-- Content -->
  </body>
</html>
```
